# Beipy-Video-player

Beipy-Video-player 用于 Web 端多功能播放器插件，适用于视频网站，添加自定义播放器功能等。

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/Beipy/Beipy-Video-player.git
```

- 安装依赖

```bash
cd Beipy-Video-player

yarn

```

- 运行

```bash
yarn serve
```

- 打包

```bash

//开发打包：
yarn build

//生产打包
yarn build:pro

```
